Exodus

Welcome.

This is my new blog, hosted on Github Pages and running on Hugo (after much fighting with inscrutable software dev nerd shit). It’s here because I got sick of Wordpress being clunky and locking useful quality of life features behind a paywall. Hopefully this way you won’t need uBlock Origin so much.

For what it’s worth, the old blog is not being deleted, just deprecated – I’ll be keeping all of my old writings there for archival purposes. If you’d like to read them again, they’re all still on the old blog.

Somewhat unfortunately, this does mean that you won’t be able to receive notifications for when new articles come up, thanks to the lack of a follow feature or some equivalent. If you’d like to stay up to date on when new articles come out, I’ll post links on my Twitter feed.

Keep your eyes peeled for more writing.
